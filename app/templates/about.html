Crypter

About

The goal of Crypter is to provide simple, anonymous, and secure file transfer to the public. Crypter does not collect your personal information or device information. Users are not tracked whatsoever.

Notice this website has "https" in the URL. That means file are uploaded to the Crypter server through a secure connection. As soon as the server receives your file, it encrypts the file using the encryption key you provide. Please use strong keys, up to 32 characters long. The encrypted file remains on the server for up to 24 hours or until someone downloads the file with the correct decryption key. Keys are NOT saved on the server. Encrypted files cannot be decrypted without the key.

To download a file you must provide the file "ID" and a decryption key. Expect an error message if the file does not exist or the decryption key could not decrypt the file.

Roadmap

This website is in early development. Please send questions and feedback to jackedwards@protonmail.com.

  • Improve the user experience, based on user feedback.
  • Fix bugs as they are reported.
  • Add a second layer of encryption, in the browser. This will require users to keep a second key which never gets sent to the server.
